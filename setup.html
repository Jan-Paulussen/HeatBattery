How to Setup your raspberry for the HeatBatt;

1. Start by installing Raspbian on your Raspberry Pi USB-key, SD-card or EMMC. This is the official operating system recommended by the Raspberry Pi Foundation. The program to use best is probably Imager. Click the little setup icon to go into the options before you write the image:
- to enable SSH if needed
- to change the hostname
- to setup the Wifi SSID and password


2. Install Apache2. You can do this by running the following command in the terminal: sudo apt-get install apache2

3. You will need to install MariaDB. You can do this by running the following command in the terminal: sudo apt-get install mariadb-server php-mysql -y

4. Once MariaDB is installed, you will need to configure it by running the following command: sudo mysql_secure_installation.
    The default username = root
    The default password for the database = raspberry

5. You will then need to install PHP. You can do this by running the following command in the terminal: sudo apt-get install php -y

6. Finally, restart Apache2: sudo service apache2 restart

7. Install the MQTT library: pip install paho-mqtt
